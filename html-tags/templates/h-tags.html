Tags <h>

Tags <h1>, <h2>, <h3>, <h4>, <h5> and <h6> are the HTML headings. They are used for article heading creation. Code example:

<body>
    <h1>Some new heading.</p>
    <h2>Some new heading.</p>
    <h3>Some new heading.</p>
    <h4>Some new heading.</p>
    <h5>Some new heading.</p>
    <h6>Some new heading.</p>
</body>

Basic HTML code

Basic HTML

This is the code for default HTML page:

<!DOCTYPE html>
<html>
<head>
    <title>Title</title>
</head>
<body>

</body>
</html>

<!DOCTYPE html> - It says to browser that this an HTML file.

<html>...</html> - Everything in this tag is an HTML code.

<head>...</head> - This tag contains information about the page. Sets tab's name, allow you to connect some scripts and styles and so on.

<title>Title</title> - Sets browser tab's name will be "Title". You can check it, if you hover the mouse cursor over opened tab in browser.

<body>...</body> - This tag contains the body of the page. That's the place, where you should start "coding" your site.

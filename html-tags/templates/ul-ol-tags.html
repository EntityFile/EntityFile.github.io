Tags <ul>, <ol>

Tags <ul>, <ol> are the HTML listings. Code example:

<body>
    <ul>
        <li>Some thing 1</li>
        <li>Some thing 2</li>
        <li>Some thing 3</li>
    </ul>

    <ol>
        <li>Some thing 1</li>
        <li>Some thing 2</li>
        <li>Some thing 3</li>
    </ol>
</body>

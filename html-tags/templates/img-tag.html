Tag <img>

Tag <img> is the HTML image. You can use this tag to insert an image to your web page. The "alt" attribute stands for text that will appear if the image would not load. Code example:

<body>
    <img src="https://miro.medium.com/max/1200/1*mk1-6aYaf_Bes1E3Imhc0A.jpeg"
    alt="Baby Yoda" width="500" height="400">
</body>

CSS text formatting

Format your text

These are some basic properties you can use to format your text elements. Example below:

p {
    text-decoration: none;
    font-family: "Bodoni MT Black", sans-serif;
    font-size: 30px;
    text-align: center;
    color: darkmagenta;
    background-color: #0f8a9d;
    padding: 30px;
}

text-decoration - adds decoration to the text. Possible values: none, underline, overline, line-through.

text-align - aligns the text in an element. Possible values: left, right, center, justify.

color - sets a color to the text.

font-family - used to change the font of the text. It consists of two parts: family-name and generic-name. family-name is the name of a font-family. For instance: "Arial", "Times New Roman". generic-name is the name of a generic-family. For example: "serif", "sans-serif", "cursive" etc.

font-size - sets a size of the text.

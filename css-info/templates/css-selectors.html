CSS selectors

CSS ID selector

There are several types of selectors in CSS. The id selector is one of them. It is used to select some appropriate id's. CSS example:

#par1 {
    color: darkgoldenrod;
    text-align: center;
    font-size: 30px;
}

We use # symbol so select appropriate id.

CSS class selector

The second type is class selector. It is used to select a specific class. CSS Example:

.par1 {
    color: lightcoral;
    text-align: right;
    font-size: 40px;
}

Your HTML page will look like this

You can also specify that only specific HTML elements should be affected by a class. In this example only <p> elements with class="par1" will be affected:

p.par1 {
    color: lightcoral;
    text-align: right;
    font-size: 40px;
}

We use . symbol so select appropriate class.

CSS universal selector

This selector selects all elements on the page. Example:

* {
    color: green;
}

CSS grouping selector

This selector selects all elements with same style definitions. Example:

h1 {
    text-align: center;
}

p {
    text-align: left;

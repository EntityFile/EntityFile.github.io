CSS syntax

Basic syntax

The syntax in CSS is very simple. See the example:

p {
    color: darkgreen;
    font-weight: 900;
}

p is a selector in CSS. It points to the HTML element you want to style.

color and font-weight are properties.

darkgreen and 900 are property values.

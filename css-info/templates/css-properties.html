CSS basic properties

Basic properties

These are some basic properties you can use to develop your site. See the example below:

body {
    background-color: lightblue;
    }

p {
    text-decoration: overline;
    text-align: center;
    color: cornflowerblue;
    background-color: #0f8a9d;
    padding: 30px;
}

background-color - basically sets a background color to an element.

text-decoration - adds decoration to the text. Possible values: none, underline, overline, line-through.

text-align - aligns the text in an element. Possible values: left, right, center, justify.

color - sets a color to the text.

padding - used to generate space around an element's content, inside of any defined borders.

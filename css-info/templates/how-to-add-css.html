How to add CSS to HTML

External CSS

There are 3 ways of adding CSS to your web page. This is the first and maybe the most common way - by linking a CSS stylesheet to your HTML code using <link> tag. Example:

<!DOCTYPE html>
<html>
<head>
    <title>Title</title>
    <link rel="stylesheet" type="text/css" href="style.css">
</head>
<body>
    <p>Just a paragraph.</p>
</body>
</html>

rel property defines what you are connecting to the HTML file.

type property defines the type of connected element.

href property defines the reference to the connected element.


Lets fill the CSS file:

body {
    background-color: forestgreen;
}

p {
    font-family: Arial, sans-serif;
    color: darkgreen;
    font-weight: 900;
}

Your HTML page will look like this

Internal CSS

The second way is adding CSS into your web page. You can do it using the <style> tag. Example:

<!DOCTYPE html>
<html>
<head>
    <title>Title</title>
    <style>
        body {
            background-color: palevioletred;
        }

        p {
            font-family: "Franklin Gothic Medium", sans-serif;
            color: purple;
            font-size: 50px;
        }
    </style>
</head>
<body>
    <p>Just a paragraph.</p>
</body>
</html>

Your HTML page will look like this

Inline CSS

The third way is adding a style property into appropriate tag. Example:

<!DOCTYPE html>
<html>
<head>
    <title>Title</title>
</head>
<body>
    <p style="font-size: 25px; color: cornflowerblue;">Just a paragraph.</p>
</body>
</html>
